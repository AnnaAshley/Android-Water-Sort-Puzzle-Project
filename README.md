# Android Water Sort Puzzle

An Android implementation of the Water Sort Puzzle featuring cutom game logic written in Java and an automatic solver capable of computing valid move sequences.

This project was developed using Android Studio as a part of a mobile development course and demonstrates object-oriented design, state modeling, and algorithmic problem solving.

## Features
- Interactive touch-based gameplay
- Automatic puzzle solver that computes valid move sequences
- Custom game logic implemented entirely in Java
- Rule-based move validation and win detection
- Puzzle reset functionality
- Android UI built using XML layouts and Activity life cycle
- Gradle Kotlin DSL build configuration (`build.gradle.kts`)

---

## How the Solver Works

The solver models the puzzle as a set of tubes states andevaluates valis pour operation according to the game rules.

Core concepts implemented:

- Representation of puzzle state using structured Java objects
- Validation of legam moves based on color matching and capacity contraints
- Simulation of moves to generate new puzzle states
- Iterative search through possible states until a solved configuration is found
- Execution of the solution sequence inside the Android application

This demonstrates:

- Algorithm design
- State-space search techniques
- Object-oriented modeling
- Complex logic implementation in Java
